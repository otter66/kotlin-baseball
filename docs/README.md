
- 입력 
    - [x] 세자리 숫자를 입력 받는다.
    - [x] 재시작 여부를 입력 받는다.

- 출력
    - [x] 게임 시작을 알리는 문구를 출력한다. "숫자 야구 게임을 시작합니다."
    - [x] 세자리 숫자 입력 요청 문구를 출력한다. "숫자를 입력해주세요 :"
    - [x] 볼과 스트라이크 개수를 출력한다. "1볼 1스트라이크"
    - [x] 성공을 알리는 문구를 출력한다. "3개의 숫자를 모두 맞히셨습니다! 게임 종료"
    - [x] 재시작 여부 입력 요청 문구를 출력한다. "게임을 새로 시작하려면 1, 종료하려면 2를 입력하세요."
